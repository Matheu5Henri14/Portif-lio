#  Portfólio Front-End | UX/UI

Este é o meu portfólio desenvolvido em **HTML e CSS**, com foco em **design de interfaces (UX/UI)**.  
O objetivo é apresentar minhas habilidades em desenvolvimento front-end e boas práticas de usabilidade.

---

##  Demonstração
🔗 [Acesse o portfólio online]https://matheu5henri14.github.io/Portifolio/  


---

##  Tecnologias utilizadas
- **HTML5**
- **CSS3** (Flexbox, Grid)
- Organização modular de estilos: `header.css`, `introducao.css`, `formacao.css`, `experiencia.css`, `global.css`, `style.css`

---

##  Estrutura do projeto
- `index.html` → Página principal
- `style.css` → Estilos globais
- `header.css`, `introducao.css`, `formacao.css`, `experiencia.css` → Estilos específicos por seção
- `img/` → Imagens utilizadas no portfólio

---

##  Funcionalidades
- Layout responsivo
- Seções: **Introdução**, **Formação**, **Experiência**, **Projetos**
- Design focado em UX/UI e navegação intuitiva
